## Análise e Modelagem de Sistemas I

**Componente Curricular**: Análise e Modelagem de Sistemas I  
**Semestre**: 3º  
**Carga horária:** 80h (67h relógio)  
**Professor**: Itamar Pena Nieradka
- - -
## Objetivo

Fornecer uma **visão geral dos processos de desenvolvimento de software**, abordando:

- Fundamentos da Engenharia de Software   
- Técnicas de análise de sistemas
- Modelagem de software orientado a objetos
- Utilização da linguagem **UML (Unified Modeling Language)**
- Elicitação e documentação de requisitos

O foco principal é compreender como planejar e estruturar sistemas antes da implementação.

---
## Ementa

- Visão geral da Engenharia de Software
- Análise e especificação de requisitos
- Processos de desenvolvimento de software (tradicionais e ágeis)
- Linguagem de modelagem unificada **UML**
- Diagramas de **Casos de Uso**
- Modelagem de **Classes** 
- Cenários de casos de uso
- Técnicas de documentação de modelagem de software
